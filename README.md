## üöÄ Installation

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server or build tools required

### Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sumiksha2058/portfolio-.git
   cd portfolio-
   ```

2. **Open the website:**
   - Simply open `index.html` in your web browser
   - Or use a local server (optional):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Python 2
     python -m SimpleHTTPServer 8000
     
     # Using Node.js (with http-server)
     npx http-server
     ```

3. **Access the website:**
   - Direct: Open `index.html` in your browser
   - Local Server: Navigate to `http://localhost:8000`

## üíª Usage

### Viewing the Portfolio
1. Open `index.html` in any web browser
2. Navigate through sections using the navigation menu
3. Click on social links to connect on GitHub or call

### Customizing the Portfolio
To customize this portfolio for your own use:

1. **Update Personal Information:**
   - Edit the name, education, and contact details in `index.html`

2. **Modify Colors:**
   - Change the color scheme in `styles.css` by updating the CSS variables:
     ```css
     :root {
         --primary-color: #667eea;
         --secondary-color: #764ba2;
         /* ... other colors ... */
     }
     ```

3. **Add New Sections:**
   - Add new HTML sections in `index.html`
   - Create corresponding CSS styles in `styles.css`
   - Add interactivity in `script.js` if needed

4. **Update Social Links:**
   - Modify the GitHub URL and phone number in the contact section
