Welcome to the Web! 👋

Web development is like building a house. HTML is the structure (the walls and beams), and CSS is the decoration (the paint and furniture).

Fun Fact: The first website ever made is still online! It was created in 1991.

HTML Basics 🏗️

HTML stands for HyperText Markup Language. It uses "tags" to tell the browser what to show.

Example Code
<h1>Hello World!</h1>
<p>This is my first webpage.</p>

HTML Structure 🦴

Every HTML file needs a specific structure to work correctly.

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <!-- Content goes here -->
  </body>
</html>

CSS Fundamentals 🎨

CSS (Cascading Style Sheets) makes your website look beautiful. You can change colors, fonts, and sizes!

h1 {
  color: #ff4757;
  font-size: 40px;
  text-align: center;
}
